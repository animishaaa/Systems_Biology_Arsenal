# Systems_Biology_Arsenal

This repository is my Systems Biology Arsenal -- a growing collection of the tools, methods, and concepts I use to study and analyze biological systems.

Here, I document detailed explanations along with reproducible R and Python code covering topics such as:

Biology
Mathematics
Statistics
Bioinformatics
Computational Biology
Systems Biology
Machine Learning
Infection Biology
Mathematical Modeling
SIR and other epidemiological models
Data Analysis and Visualization

The goal of this repository is not just to collect code, but to explain the underlying concepts, demonstrate practical implementations, and build a reusable toolbox for biological data analysis and systems-level thinking.
